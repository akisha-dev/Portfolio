# AKANKSHA_OS V98

**[→ Live Demo](https://akisha-dev.vercel.app)**

A Windows 98-themed developer portfolio built from scratch in React.
No UI kits. No templates. Custom everything.

## What's inside

| Feature | Details |
|---|---|
| Boot sequence | Terminal-style loading screen with typed lines and a progress bar |
| OS chrome | Custom window titlebar, menu bar, and taskbar with a live clock |
| File system nav | React Router mapped to `C:\AKANKSHA\` — About, Projects, Contact |
| Responsive sidebar | Collapses to a top nav bar on mobile |
| Folder icons | Pure CSS, no image assets |

## Stack

- React + Vite
- Tailwind CSS v4
- React Router DOM

## Run locally

```bash
git clone https://github.com/akisha-dev/Portfolio.git
cd Portfolio
npm install
npm run dev
```

## Projects featured

- **GitCard** — GitHub profile → shareable PNG card (live API + html2canvas)
- **Keyboard Heatmap** — real-time keystroke heatmap in pure JS
- **TypingFlow** — WPM typing test, no libraries
